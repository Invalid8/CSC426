# CSC426 — Practical Works & Assignments

A repository containing practical works, assignments, and projects for **CSC426**.

---

## Projects

### 1. Calculator
A GUI-based calculator built with HTML, CSS, and JavaScript.

**Features:**
- Supports `+`, `-`, `*`, `/`, `\` (integer division), `^`, `%` operations
- Clear (`C`) button to reset
- Expression history display
- Keyboard support
- Responsive and clean UI

**Live Demo:** https://Invalid8.github.io/CSC426/calculator/

**Source Code:** [/calculator](/calculator)

---

### 2. Login App
A web-based login authentication application built with HTML, CSS, and JavaScript.

**Features:**
- Username and password fields
- Input validation with error messages
- Success and error feedback
- Show/hide password toggle
- Reset/Cancel button
- Responsive UI
- localStorage-free — validation handled entirely in JS

**Live Demo:** https://Invalid8.github.io/CSC426/login-app/

**Source Code:** [/login-app](/login-app)
